# Netflix Data Cleaning Project

## Objective
The goal of this project is to clean and preprocess the Netflix Movies and TV Shows dataset using Microsoft Excel.

## Dataset
- Raw dataset: Raw_Data.xlsx
- Source: Netflix Movies and TV Shows dataset

## Cleaning Steps Performed
- Identified and handled missing values
- Standardized text fields using TRIM and PROPER
- Fixed date format inconsistencies
- Removed duplicate records
- Documented data quality issues
- Separated raw and cleaned datasets

## Files Included
- Raw_Data.xlsx (raw data)
- Cleaned_dataset.xlsx (cleaned Excel file)
- cleaned_dataset.csv (exported CSV)

## Data Quality Notes
- Missing director values replaced with 'Unknown'
- Missing cast values replaced with 'NA'
- Duration values mixed between minutes and seasons
- Character encoding issues present in some names
- Date formats standardized

## Tools Used
- Microsoft Excel
- GitHub
